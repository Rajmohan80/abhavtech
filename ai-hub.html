Why "Small" Language Models?

Each tool here is scoped deliberately — tight prompts, retrieval-grounded context, or a fine-tuned model trained on domain data. Narrow scope means higher accuracy and lower cost for the engineering problems that matter.

Cost-First by Design

Every tool is built with a zero-API-key-in-the-browser rule. Edge proxies hold secrets. Rate limits and token caps bound the cost. The SLM for Webex demo is scripted — zero API calls, zero cost to share.

Demonstration scope. All tools are AI-assisted demos. Validate independently before any production use — see the disclaimer on the main portfolio page.
