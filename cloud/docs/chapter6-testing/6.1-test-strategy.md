# Document Purpose


This guide is the capstone test document for the entire multi-cloud deployment. It validates every integration established across Guides 1–4:

- **Guide 1 deliverables:** GCP Vertex AI, Cloud Interconnect, BGP to Mumbai hub
- **Guide 2 deliverables:** Azure ExpressRoute (3 circuits), Virtual WAN (19-site IPsec), Private Endpoints
- **Guide 3 deliverables:** SD-WAN Cloud OnRamp SaaS/IaaS/SASE, Umbrella, QoS
- **Guide 4 deliverables:** vManage templates, VPN segmentation, SD-Access handoff, ZTP

**Test Philosophy:** Every test has a defined pass criterion, a failure action, and a rollback path. Tests progress from component-level (Week 1) through integration (Week 2) to chaos/failover testing (Week 3) and formal user acceptance (Week 4).
