- Business case development

## Migration Patterns

**Greenfield Deployment:** New sites deployed directly with SD-WAN  
**Brownfield Migration:** Gradual replacement of existing WAN infrastructure  
**Hybrid Operation:** Parallel operation during transition  
**Big Bang Cutover:** Coordinated simultaneous migration (high risk)  
**Phased Rollout:** Incremental migration by site tier (recommended)
