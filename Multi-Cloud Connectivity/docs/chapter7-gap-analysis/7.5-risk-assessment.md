## Multi-Cloud Transition Path


**Recommended Sequence:**
1. **Months 1-4**: GCP (Vertex AI + WxCC) ← **Current Focus**
2. **Months 5-7**: Add AWS (enterprise apps, data lake)
3. **Months 8-10**: Add Azure (Office 365, Azure AD)
4. **Month 11+**: Optimize (inter-cloud routing, cost management)

**Connectivity Strategy:**
- Start with **Hub-and-Spoke** (all clouds → on-prem → other clouds)
- Upgrade to **Hybrid** if latency becomes issue (direct GCP-AWS for Vertex AI → S3)
- Avoid full **Mesh** unless >50% traffic is inter-cloud
